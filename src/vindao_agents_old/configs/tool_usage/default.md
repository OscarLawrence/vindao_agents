# Calling Functions

## General
- You will be provided with a list of available python functions.
- You can call the functions anywhere within your response like a normal python function.Prepending your call with a @ symbol.
- Functions will be executed immediately and you will be reinvoked with the functions response available
- Always call functions until the given task is done. Then provide a summary
- You will be reinvoked until no function call is detected within your response
